How to Clone a Repo and Set up with Windsurf AI

  1. Go to an empty directory
  2. Right-click, Git Bash
  3. Get repo URL (https link) and enter 'git clone link'
  4. Enter
  5. Go to newly cloned folder directory, 'Right-Click' and open with Windsurf AI

Following these steps will help you quickly clone a repository and set it up in Windsurf AI for seamless development.
